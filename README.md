# 托业学习助手

一站式托业(TOEIC)英语学习平台，纯前端实现，数据存储在本地浏览器，支持电脑和手机访问。

## 功能概览

### 1. 单词记忆 (`vocabulary_app.html`)

- **词库导入**：支持上传自定义 JSON 词库文件，也可一键加载内置示例数据
- **卡片翻转**：正面显示单词和词性，点击翻转查看中文释义、近义词、反义词、用法、例句等
- **学习标记**：对每个单词标记「认识」「不认识」「收藏」，数据持久化到 localStorage
- **筛选模式**：只看认识 / 只看不认识 / 只看收藏 / 全部显示
- **搜索功能**：按单词、释义、词性实时模糊搜索
- **学习进度**：顶部进度条显示当前学习位置和已认识数量
- **内置词库**：`word_database.dat` 包含 1000+ 托业核心词汇

### 2. 题库练习 (`practice_coach.html`)

- **AI 智能出题**：基于火山引擎大模型 API 实时生成托业 Part 5 语法/词汇单选题
- **本地题库**：支持导入/导出 JSON 格式题库，离线也可刷题
- **题目池机制**：预生成题目池，下一题无需等待 API 调用
- **智能去重**：通过题干哈希记录已出题目，避免重复
- **多维度解析**：每题提供中文翻译、核心词汇、解题步骤、逐选项分析、考点小结
- **难度选择**：L1 ~ L5 五档难度
- **学习统计**：总练习数、正确率、错题数、按考点类别统计
- **错题回顾**：按全部/仅错题/仅正确筛选，点击展开详细解析

### 3. 模拟考试 (`practice_coach.html` → 考试页)

- **题库出题**：从本地题库随机抽取 30 题，无需 API Key
- **限时考试**：10 分钟倒计时，到时自动交卷
- **答题导航**：可点击题目序号跳转，已答/未答状态一目了然
- **AI 考后分析**：交卷后 AI 自动分析薄弱考点、进步趋势、学习建议
- **成绩曲线**：Canvas 绘制历次考试正确率趋势图
- **历史记录**：每次考试记录持久化到 IndexedDB，可查看历次成绩和趋势

### 4. 词汇故事 (`vocab_story.html`)

- **故事串联**：通过 5 篇关联文章串联托业高频词汇，覆盖商务、职场、法律、科技、日常生活等场景
- **点击查词**：文章中高亮词汇可点击查看详细释义、词性、近义词/反义词
- **内置词库**：包含 1042 个托业核心词汇
- **搜索功能**：顶部搜索栏可快速查找单词

### 5. 批量题目生成器 (`question_batch_generator.html`)

- 批量生成托业 Part 5 题目并保存到本地题库
- 支持选择生成数量和难度
- 生成的题目可直接在刷题和考试中使用

## 技术栈

- 纯 HTML + CSS + JavaScript，无需构建工具
- 数据存储：localStorage（单词进度）+ IndexedDB（练习记录、考试历史、题目哈希）
- AI 功能：火山引擎 ARK API（模型 `ep-20260505172500-db99v`）
- 单词数据库：本地 JSON 文件，1000+ 托业核心词汇

## 快速开始

### 1. 启动服务器

```bash
python local_test.py
```

启动后会显示访问地址：
- 电脑：`http://localhost:8000/index.html`
- 手机：`http://[电脑IP]:8000/index.html`（需同一 Wi-Fi）

> 如果端口 8000 被占用，脚本会自动尝试 8080、3000、5000、9000。

### 2. 电脑端测试

浏览器打开 `http://localhost:8000/index.html`，即可看到主页，点击卡片进入各功能模块。

### 3. 手机端测试

1. 确保手机和电脑连接同一 Wi-Fi
2. 电脑端查看 IP 地址：Windows 运行 `ipconfig`，找到 IPv4 地址
3. 手机浏览器访问 `http://[电脑IP]:8000/index.html`

## 使用教程

### 单词记忆

1. 进入「单词记忆」页面，首次使用点击「加载示例数据」导入内置词库，或上传自己的 JSON 词库文件
2. 左右滑动或点击底部「学习」「搜索」「词库」切换页面
3. 点击卡片翻转查看释义，点击「认识」「不认识」「收藏」标记状态
4. 点击「只看认识」「只看不认识」「只看收藏」筛选复习
5. 顶部搜索栏可按单词、释义、词性实时搜索

### 刷题练习

1. 进入「题库练习」页面，默认使用「本地题库优先」模式
2. 如果本地题库有题，会自动加载；如果没有，切换到「AI 生成」模式
3. 选择难度和预生成数量，点击「新题目」出题
4. 选择答案后点击提交，查看详细解析（翻译、核心词汇、解题步骤、选项分析、考点小结）
5. 底部「记录」页可查看所有做题记录，支持筛选仅错题/仅正确
6. 底部「设置」页可管理 API Key、导入/导出题库、清空缓存

### 模拟考试

1. 进入「题库练习」页面，点击底部「考试」
2. 查看考试说明，点击「开始考试」
3. 从本地题库随机抽取 30 题，10 分钟倒计时
4. 可点击题目序号跳转答题，已答/未答状态实时显示
5. 最后一题点击「提交考试」，或时间到自动交卷
6. 查看考试结果：正确率、分类统计、成绩趋势图
7. AI 自动分析薄弱考点和学习建议（需已配置 API Key）
8. 点击「题目回顾」展开每题的详细解析
9. 点击「查看历史记录与成绩曲线」查看历次考试趋势

### 词汇故事

1. 进入「词汇故事」页面
2. 选择一篇故事阅读，文中高亮词汇为托业核心词
3. 点击高亮词弹出详细释义面板，查看词性、近义词、反义词
4. 顶部搜索栏可快速查找单词
5. 适合在刷题之余通过语境加深词汇记忆

### 批量生成题目

1. 进入「批量生成」页面（需已配置 API Key）
2. 选择生成数量和难度
3. 点击生成，AI 批量出题并保存到本地题库
4. 生成的题目可在刷题和考试中直接使用

## API Key 配置

AI 出题和 AI 考后分析功能需要火山引擎 ARK API Key：

1. 打开「题库练习」→ 底部「设置」
2. 在 API Key 输入框填入你的 `ARK_API_KEY`
3. 点击保存

> 单词记忆、词汇故事、模拟考试（题库出题）无需 API Key。

## 词库 JSON 格式

```json
[
  {
    "word": "increase",
    "pos": ["名", "动"],
    "meaning": "增加",
    "synonyms": ["surge激增"],
    "antonyms": ["decrease"],
    "usage": "be expected to do",
    "notes": "考试高频词",
    "related": "相关词",
    "examples": ["例句1"]
  }
]
```

其中 `word`、`pos`、`meaning` 为必填字段，其余可选。
