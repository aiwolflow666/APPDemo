# 托业学习助手 📚

一站式托业英语学习平台，包含单词记忆和题库练习功能。

## 🚀 功能特点

### 📖 单词记忆
- 系统学习2000+托业核心词汇
- 包含词性、中文释义、同反义词
- 支持单词卡片、测试模式
- 完全本地运行，无需网络

### 🧠 题库练习
- 智能生成托业Part 5语法/词汇题目
- 多种难度级别可选
- 详细答案解析和解题步骤
- 学习统计和错题回顾
- 题目去重机制，避免重复

## 📱 手机访问

1. **获取电脑IP地址**:
   - Windows: `ipconfig` (查找IPv4地址)
   - Mac/Linux: `ifconfig` 或 `ip addr`

2. **确保在同一网络**:
   - 手机和电脑连接同一个Wi-Fi

3. **访问地址**:
   ```
   http://[电脑IP地址]:8000/index.html
   ```
   例如: `http://192.168.1.100:8000/index.html`

## 🔍 常见问题

### Q: 题库练习需要API Key吗？
**A**: 是的，题库练习需要火山引擎API Key来生成题目。如果没有API Key，可以：
1. 测试界面和本地功能
2. 使用单词记忆功能（完全本地）
3. 联系获取测试用API Key

### Q: 题目仍然重复怎么办？
**A**: 
1. 点击"清空题目缓存"按钮
2. 重新生成题目池
3. 确保单词数据库加载正常

### Q: 手机无法访问？
**A**: 
1. 检查电脑防火墙设置
2. 确保手机和电脑在同一网络
3. 检查IP地址是否正确
4. 尝试使用不同的端口
