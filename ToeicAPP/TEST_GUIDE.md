# 托业学习助手 - 本地测试验证指南

## 📋 测试前准备

### 1. 确认文件完整性
运行以下命令检查所有文件：
```bash
python simple_test.py
```

预期输出：
- [OK] 所有文件存在
- [OK] 单词数据库加载成功 (1043个单词)
- [OK] 所有优化已应用

### 2. 启动本地服务器
**方法A：使用批处理脚本 (Windows)**
```bash
双击 quick_test.bat
```

**方法B：手动启动**
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000

# PHP
php -S 0.0.0.0:8000
```

### 3. 获取访问地址
1. **电脑测试**：`http://localhost:8000/index.html`
2. **手机测试**：`http://[电脑IP]:8000/index.html`

获取电脑IP：
- Windows：`ipconfig` (查找IPv4地址)
- Mac/Linux：`ifconfig` 或 `ip addr`

## 🧪 测试步骤

### 第一阶段：统一入口测试
1. **访问主页面**
   - 打开 `http://localhost:8000/index.html`
   - 检查页面是否正常加载
   - 验证响应式设计（调整窗口大小）

2. **功能导航测试**
   - 点击"单词记忆"卡片 → 应跳转到 `vocabulary_app.html`
   - 点击"题库练习"卡片 → 应跳转到 `practice_coach.html`
   - 测试浏览器返回按钮功能

### 第二阶段：单词记忆功能测试
1. **基本功能测试**
   - 单词卡片是否显示
   - 翻页功能是否正常
   - 测试模式是否工作

2. **数据验证**
   - 检查单词数量显示
   - 验证词性、释义显示
   - 测试同反义词显示

### 第三阶段：题库练习功能测试
1. **界面加载测试**
   - 页面是否正常加载
   - 所有按钮和输入框是否显示
   - 统计信息是否初始化

2. **API Key 测试** (可选)
   - 输入测试API Key
   - 点击"保存"按钮
   - 验证localStorage保存

3. **题目生成测试** (需要真实API Key)
   - 选择难度级别
   - 选择题目数量
   - 点击"重置池子并开始"
   - 观察题目生成过程

4. **题目多样性验证** (核心测试)
   - 连续做10-15道题
   - 观察题目是否重复
   - 检查不同场景和考点
   - 验证选项随机性

5. **去重功能测试**
   - 点击"清空题目缓存"
   - 重新生成题目
   - 验证是否允许重复题目

6. **学习统计测试**
   - 提交正确答案 → 正确率应更新
   - 提交错误答案 → 错题数应更新
   - 检查记录列表更新

## 🐛 常见问题排查

### 问题1：页面无法加载
**可能原因**：
- HTTP服务器未启动
- 端口被占用
- 防火墙阻止

**解决方案**：
```bash
# 检查端口占用
netstat -ano | findstr :8000

# 更换端口
python -m http.server 8080
```

### 问题2：题目仍然重复
**可能原因**：
- API Key无效或额度不足
- 单词数据库未加载
- 去重机制未生效

**解决方案**：
1. 检查控制台错误信息
2. 点击"清空题目缓存"
3. 重新生成题目池

### 问题3：手机无法访问
**可能原因**：
- IP地址错误
- 防火墙设置
- 网络隔离

**解决方案**：
1. 确认电脑和手机在同一网络
2. 关闭电脑防火墙临时测试
3. 使用 `ping [电脑IP]` 测试连通性
